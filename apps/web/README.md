# Frontend: SPA (Single Page Application)

SPA que consumirá la API de películas, definiendo la interfaz visual y atractiva que verá el usuario

## ⚒️ Stack: React + Typescript, orquestado con Vite

## ⚙️ Ejecución

Para probar el código y ver su resultado, ejecuta este comando en tu terminal:

```bash
 # Forma corta
pnpm dev
# Forma larga
pnpn run dev 
# Ambos funcionan perfectamente
```
